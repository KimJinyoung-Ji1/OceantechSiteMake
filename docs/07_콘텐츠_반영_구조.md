## 데이터 구조 개편 원칙
- `constants.ts`는 회사/연락처/인증 메타 중심으로 유지
- 대량 텍스트와 페이지 카피는 `i18n`으로 이동
- 기사/인증/특허/시험성적서는 배열 데이터로 정리
- 섹션 내부 로컬 상수는 가능하면 `lib` 레이어로 승격
