# 08. 디자인 리디자인 계획

## 목표
- 기존 사이트의 뼈대는 유지하되, 시각 퀄리티를 기관급 신뢰감 + 친환경 기술 기업 톤으로 끌어올린다.
- 수협 레퍼런스의 넓은 2단 헤더 구조를 차용하되, 오션테크에 맞는 더 현대적인 카드/그리드/비주얼 시스템으로 재구성한다.

## 고정 원칙
- 색상은 아래 4개만 핵심축으로 사용:
  - `#021097`
  - `#0168EF`
  - `#03E9F8`
  - `#17E9B5`
- 차콜/샌드/베이지 톤 사용 금지
- 라이트 모드 전용

## 레이아웃 방향

### 공통
- `max-width`를 더 넓히고 풀폭 섹션 적극 사용
- 제목/핵심 카피는 중앙 정렬 우선
- 섹션 구분마다 바다/항구/실증 현장 이미지 삽입
- 카드 라운드/보더/그라디언트 질감을 통일

### 헤더
- 상단 유틸바 + 하단 메인 내비 2단 구조
- 메인 메뉴는 풀폭 사용
- hover 시 드롭다운을 카드형으로 확장
- 인증현황에는 green dot 유지

### 히어로
- 좌측: 슬로건, 설명, CTA, 핵심 수치
- 우측: 실제 녹색인증 카드
- 배경은 브랜드 맞춤 해양 비주얼

### 인증/기술 카드
- 아이콘 품질 상향
- 카드 정보 밀도 증가
- 원본 보기 CTA를 더 명확하게 표현
- 썸네일 비율 통일

### 서브페이지
- `PageHero`를 페이지별 전용 분위기로 강화
- 회사소개/연혁/비전은 중앙 정렬 기반
- 기술/인증은 카드 갤러리형 정보 구조

### CTA/푸터
- CTA는 큰 카피 + 2버튼 구조 유지
- 푸터는 회사정보 / 바로가기 / 기술·인증 / 인증 배지 구조로 정리

## 우선 순위
1. 텍스트/콘텐츠 정상화
2. 헤더/푸터/히어로 리디자인
3. 홈 섹션 밀도 강화
4. 기술력/인증현황 페이지 확장
5. 회사소개/친환경/문의하기 정리
