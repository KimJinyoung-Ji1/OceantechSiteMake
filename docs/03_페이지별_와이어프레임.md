## 1. 홈페이지 (`/`)

### 1-1. 메가 헤더 (공통)
- 투명→글래스모피즘 전환
- 메뉴 호버 시 카드형 서브메뉴 드롭다운
- 녹색인증 메뉴에 green dot

### 1-2. 히어로 (Split Layout)
```
┌──────────────────────┬──────────────────────┐
│                      │                      │
│  [녹색기술인증 배지]   │                      │
│                      │    (바다 이미지)       │
│  바다를 지키는         │                      │
│  친환경 기술           │   [글래스 카드]        │
│                      │   "친환경 아연 어장추"  │
│  설명 텍스트...        │                      │
│                      │   [녹색기술인증 배지]   │
│  [기술력 알아보기] [회사소개]                    │
│                      │                      │
│  6건  10년  235종  80%│                      │
└──────────────────────┴──────────────────────┘
```
- 좌: 텍스트 + CTA + 핵심 통계
- 우: 바다 사진 + 오버레이 + 글래스 카드
- 녹색기술인증 배지가 좌측 상단에 자연스럽게 배치

### 1-3. 녹색인증 스트립 (Green Cert Bar)
```
┌─────────────────────────────────────────────┐
│ 🌿  녹색기술제품 확인서 취득                    │
│     인증번호 GT-25-02356 · 유효 2025~2029    │
└─────────────────────────────────────────────┘
```
- 히어로 직하단, 풀와이드
- 녹색인증 마크 이미지 사용 (제공된 PNG)
- 하단 보더 민트색 강조

### 1-4. WHY OCEANTECH (핵심가치 벤토 그리드)
```
┌──────────┬──────────┬──────────┐
│ 🌊       │ 📋       │ ⏱️       │
│ 친환경   │ 특허     │ 10년     │
│ 소재     │ 6건 보유 │ 내구성   │
├──────────┴──────────┴──────────┤
│ 수협 중앙회 단가계약 체결        │
│ 31개월 시범 | 19척 | 1억 규모   │
└──────────────────────────────┘
```
- 3+1 벤토 카드 (3개 작은 카드 + 1개 풀폭 카드)
- 호버 시 translateY + shadow 증가

### 1-5. 바다 디바이더
- 풀폭 바다 이미지 (400px 높이)
- 상하 그라디언트 오버레이

### 1-6. 아연추 vs 납추 비교 (미니버전)
- 핵심 3~4개 항목만 (상세는 친환경 페이지)
- "자세히 보기 →" 링크

### 1-7. 인증·수상 현황 (4카드)
- 녹색기술인증(핵심 배지) / 벤처인증 / 우수특허 / 연구소
- 카드 클릭 → 인증현황 페이지 이동

### 1-8. CTA 섹션
```
┌─────────────────────────────────┐
│   함께 만드는 깨끗한 바다          │
│   [문의하기 →]                    │
└─────────────────────────────────┘
```
- 블루→시안 그라디언트 배경
- 장식용 원형 blur 요소

### 1-9. 푸터 (공통)
- 4컬럼: 회사정보 / Company / Technology / Support
- 인증 로고 아이콘 나열
- 저작권 + 개인정보처리방침

## 2. 회사소개 (`/about`)

### 2-1. 페이지 히어로
- 배경: 항구/바다 이미지 + 오버레이
- 제목: "회사소개"
- 브레드크럼: 홈 > 회사소개

### 2-2. CEO 인사말
- 좌: CEO 사진 (또는 회사 이미지)
- 우: 인사말 텍스트
- **PM 확인 필요**: CEO 인사말 텍스트, CEO 사진

### 2-3. 비전 & 미션
- 카드 3개: 비전 / 미션 / 핵심가치
- 아이콘 + 제목 + 설명

### 2-4. 연혁 타임라인
- 세로형 타임라인 (좌우 교차)
- **PM 확인 필요**: 상세 연혁 데이터
- 현재 파악된 것:
  - 2023.12: 벤처기업 인증
  - 2023.12.27: 대한민국 우수특허 대상
  - 2025: 녹색기술제품 확인서
  - 수협 단가계약 체결
  - 고성군 시범사업 (31개월)

### 2-5. 회사 개요
- 표 형태: 회사명, 대표, 설립일, 소재지, 사업분야
- **PM 확인 필요**: 설립일, 사업자번호, 직원수 등

## 5. 친환경 (`/sustainability`)

### 5-1. 페이지 히어로
### 5-2. 왜 아연인가 (인트로)
- 납의 해양 오염 문제 → 아연의 친환경성

### 5-3. 납추 vs 아연추 비교표
| 항목 | 아연추 | 납추 |
|------|--------|------|
| 재질 | 아연(Zn) | 납(Pb) |
| 독성 | 없음 | 있음 |
| 내구성 | 약 10년 | 평균 2년 |
| 무게변화 | 30개월 1g 손실 | 연 15~20% 감소 |
| 형상유지 | 밟혀도 원형 유지 | 충격에 취약 |
| 그물손상 | 없음 | 파편으로 훼손 |
| 관리비용 | 80% 이상 절감 | 교체비 지속 |

### 5-4. SVHC 235종 불검출 강조
- 큰 숫자 "235" + 설명
- EU REACH 기준 준수

### 5-5. 환경 영향 인포그래픽
- 아이콘 기반 시각화
- 해양 생태계 보호 메시지
