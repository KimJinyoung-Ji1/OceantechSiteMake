## Supabase 스키마

### 프로젝트: 별도 생성 (PM 요청)
- Organization: 기존 지일 org 하위 또는 새로 생성

### 테이블

```sql
-- 문의 접수
CREATE TABLE inquiries (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  email TEXT NOT NULL,
  phone TEXT,
  inquiry_type TEXT NOT NULL,
  content TEXT NOT NULL,
  is_read BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- RLS 정책
ALTER TABLE inquiries ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow anonymous insert" ON inquiries FOR INSERT WITH CHECK (true);
CREATE POLICY "Allow authenticated read" ON inquiries FOR SELECT USING (auth.role() = 'authenticated');
```

## i18n 구현

```
middleware.ts → locale 감지/리다이렉트
                 ↓
app/[locale]/layout.tsx → locale 파라미터 수신
                 ↓
useTranslation(locale) → ko.json / en.json 로드
```

- `Accept-Language` 헤더로 초기 언어 감지
- 쿠키에 선택 언어 저장
- `KR / EN` 토글로 전환
