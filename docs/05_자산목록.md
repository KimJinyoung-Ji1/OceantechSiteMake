## PDF → 이미지 변환 계획

모든 PDF 인증서를 **PNG 이미지로 사전 변환**하여 `public/documents/certs/`에 배치

### 변환 작업
1. PyMuPDF(fitz)로 PDF 각 페이지를 고해상도 PNG로 렌더링
2. 인증서 영역 자동 크롭 (여백 제거)
3. 동일 비율(4:3 또는 원본 비율)로 리사이즈
4. 썸네일 (400px 너비) + 원본 (1200px 너비) 2벌 생성
5. WebP 변환으로 용량 최적화

### 파일명 규칙
```
green-tech-cert.png          # 녹색기술인증서
green-product-cert.png       # 녹색기술제품확인서
venture-cert.png             # 벤처인증서
patent-award-2023.png        # 우수특허 대상
research-dept.png            # 연구전담부서
patent-10-2197917.png        # 특허등록증
svhc-report-p1.png           # SVHC 보고서 1페이지
test-report-tak-p1.png       # TAK 시험성적서 1페이지
test-report-tbk-p1.png       # TBK 시험성적서 1페이지
```
