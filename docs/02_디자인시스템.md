## 디자인 톤 & 무드
- **친환경 + 기술력 = 깨끗하고 신뢰감 있는 톤**
- 배경: 밝은 화이트 기반 + 블루/시안 그라디언트 섹션
- 라운드: `rounded-xl` (12px) ~ `rounded-2xl` (16px)
- 그림자: 카드 기본 `shadow-lg`, 호버 시 `shadow-xl` + `translateY(-4px)`
- 여백: 섹션 간 `py-20` 이상
- **라이트 모드 전용** (다크모드 미지원)

## 헤더 리디자인 (PM 요구사항 반영)

### 기존 확정시안
- floating pill 스타일 내비, 56px 높이
- 로고 + 6개 링크 + 언어토글 + CTA 버튼이 한 줄에 조밀하게 배치

### 개선안: **풀와이드 메가 헤더**
```
┌─────────────────────────────────────────────────────────┐
│  [로고]           홈  회사소개  기술력  인증현황  친환경  │  KR/EN  [문의하기]
│                                                         │
│  (마우스 오버 시 → 서브메뉴 드롭다운 슬라이드 다운)       │
│  ┌─────────────────────────────────────────────────────┐ │
│  │  ● CEO 인사말    ● 비전    ● 연혁                    │ │  ← 회사소개 호버 시
│  │  (카드형 미리보기 + 아이콘 + 간단 설명)              │ │
│  └─────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

**특징:**
1. 스크롤 전: 투명 배경 + 화이트 텍스트 (히어로 위에 올라감)
2. 스크롤 후: 글래스모피즘 (blur + 반투명 화이트)
3. 메뉴 항목 호버 시: 부드러운 서브메뉴 드롭다운 (Framer Motion `AnimatePresence`)
4. 서브메뉴: 카드형 레이아웃 (아이콘 + 제목 + 한줄 설명)
5. 녹색인증 메뉴에 초록색 dot indicator
6. 모바일: 햄버거 → 풀스크린 드로어 (슬라이드 in)
7. 전체적으로 여유로운 간격, 조밀하지 않은 배치

## 애니메이션 시스템
| 트리거 | 효과 | 라이브러리 |
|--------|------|-----------|
| 페이지 진입 | 페이드인 + 슬라이드업 | Framer Motion |
| 스크롤 진입 | 순차 페이드인 (stagger) | Framer Motion `whileInView` |
| 카드 호버 | scale(1.02) + shadow 증가 | Tailwind transition |
| 헤더 메뉴 호버 | 서브메뉴 슬라이드 다운 | Framer Motion `AnimatePresence` |
| 숫자 통계 | 카운트업 (0→목표값) | Custom hook |
| CTA 버튼 | pulse glow 효과 | CSS animation |
